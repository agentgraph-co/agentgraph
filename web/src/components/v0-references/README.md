# v0 Style-Matched Component References

These files are **reference designs** generated by v0 AI to extract styling ideas from.
They are NOT drop-in replacements — adapt the patterns to match existing component APIs
and AgentGraph's Tailwind v4 @theme patterns.

## Files

1. `PostCard-reference.tsx` — Atmospheric post card with bioluminescent border glow
2. `ProfileHeader-reference.tsx` — Profile header with gradient mesh background
3. `StatCard-reference.tsx` — Dashboard stat card with teal glow and count-up animation

## Key Patterns to Extract

### Bioluminescent Border Glow (PostCard)
- Animated gradient border using `mask-composite: xor` technique
- `background-size: 300% 300%` with `background-position` keyframes
- Border shifts between teal (#0D9488) and fuchsia (#E879F9)

### Gradient Mesh Background (ProfileHeader)
- Multiple `blur-3xl` orbs with brand palette colors
- Staggered `animate-pulse` with delay classes
- Glass morphism: `backdrop-blur-xl bg-[#111B1B]/40`

### Animated Glow Border (StatCard)
- Outer glow: `absolute -inset-0.5 blur-sm animate-pulse`
- Inner glow on hover: `bg-gradient-to-br from-primary/5 via-transparent to-primary-light/5`
- Teal shadow: `shadow-[#0D9488]/10` intensifying on hover

### Count-Up Animation (StatCard)
- `requestAnimationFrame` with easeOutQuart easing
- Tabular nums for stable width: `tabular-nums tracking-tight`
